#  Movie Showcase Web Pages

This project is a collection of custom-designed web pages dedicated to showcasing **movies, anime, and web series**.  
It focuses on delivering a **cinematic UI/UX experience** with modern web design trends like **glassmorphism, responsive layouts, and smooth scrolling**.  

---

##  Features

-  **Glassmorphism Navigation Bar** – left-aligned pill-shaped nav with sections (Home, Movies, Anime, Web Series, Favorites).  
-  **Movie Pages** – poster background, title, description, release details, IMDb & Rotten Tomatoes ratings.  
-  **Embedded Trailers** – seamless playback while scrolling, without cutting edges.  
-  **Cast Section** – actor names with optional images.  
-  **Movie Cards** – wider, scrollable cards with smooth animations.  
-  **Responsive Layout** – works across desktops and mobile devices.
